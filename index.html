Quick Reference Guide

PHC Resident Doctors Activity Tracker

Department of Community Health & PHC · Lagos State

📋 Duty Roster

Records where each doctor is posted each month.

  • Click + Posting (top bar) to add a new entry
  • Set Designation: SR, R, or MO
  • Update Report Status to Signed once the consultant signs off
  • Filter by Year, Month, Station, Designation, or Report status
  • Use ⬇ CSV to export the full roster
🎯 Target: ≥ 6 signed consultant reports per doctor per year

🗓 Leave Schedule

Tracks all planned leave with live status.

  • Click + Leave (top bar) to add a leave entry
  • Enter start & end dates, type, and reliever doctor
  • Status updates automatically: On leave / Upcoming / Completed
  • Filter by Year, Leave Type, Status, or Designation
💡 Reliever field helps identify coverage for absent doctors

📖 Journal & Seminar Presentations

Logs scheduled and completed presentations.

  • Click + Activity → select Journal Presentation or Seminar
  • Fill in Title, Date, Venue, and Discussant/Chairperson
  • Update Status: Scheduled → Completed or Missed
🎯 Target: ≥ 1 journal presentation + ≥ 1 seminar per doctor per year

✅ Attendance Register

Tracks which doctors attended each session.

  • Click + Record Attendance on the Attendance page
  • Select the session, then mark each doctor: Present / Absent / Excused
  • The matrix view shows attendance across all sessions at a glance
  • Filter by session type, year, or individual session
💡 Attendance rate is shown per doctor and per session

🔬 Research & Publications

Tracks research projects and publications.

  • Click + Activity → select Research / Publication
  • Select type: Research project, Journal publication, Conference paper, Thesis, or Book chapter
  • Status: In progress → Submitted → Published
🎯 Target: ≥ 1 publication + ≥ 1 research project per doctor per year

📊 Statistics & Performance

Shows performance scores based on annual targets.

  • Each of the 5 targets is worth 20 points (max score = 100)
  • Filter by Year and Designation (SR / R / MO)
  • Use ⬇ Export report to download as CSV
  • Leaderboard ranks doctors by overall score
✓ Green = target met   ✗ Red = below target

Go to Import Data in the sidebar. Select the data type tab, then upload a CSV file.
